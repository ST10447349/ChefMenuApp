This is part 3 of my POE: Menu items are saved using AsyncStorage in the AddMenuScreen. Each item gets a unique ID and the price is stored as a number. The HomeScreen loads the saved items and sends them to the FilterScreen. The FilterScreen shows and filters the menu without reading from storage again, making the app faster and simpler.
